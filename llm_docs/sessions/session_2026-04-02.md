### design decisions made today

- jet / air-breathing parts should **not** be penalized during rocket generation
- structural validity should remain unchanged
- the correction belongs in evaluation, specifically inside the delta-v calculation
- we do **not** want a hard filter that bans jets
- preferred direction is a rough stage-by-stage ascent estimate rather than a fixed dv cap
- chosen fidelity: account for changing mass during a stage, but keep simplifying assumptions
- acceptable simplifying assumptions for this notebook phase:
  - vertical ascent
  - average mass over the stage
  - no pitch program
  - no detailed drag model yet
  - conservative approximation is preferred over optimistic over-credit

### implementation direction

- keep `compute_delta_v(...)` as the main orchestrator
- add a helper that takes the current stage state and returns:
  - stage contribution
  - ending mass
  - ending velocity
  - ending altitude
- use that helper inside the stage loop to handle both:
  - normal rocket stages
  - air-breathing stages with a ceiling-limited contribution

  - so the filter is over-crediting upper-stage engines that do nothing for launch viability
