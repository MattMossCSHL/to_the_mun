## design decisions locked in

- **no filtering in the generator** — rockets are generated randomly with no validation. bad designs score 0, GA learns from zero. revisit only if generation 0 is all zeros.
- **fitness = delta-v or 0** — validate → filter → compute_delta_v. cliff at 0, gradient above it.
- **selection = tournament** — not fitness-proportionate. more robust, preserves diversity.
- **crossover = stage-level** — upper stages from parent A, lower from parent B.

## what's next

**`tournament_select`** — the selection function.

signature:
```python
tournament_select(population, scores, n_survivors, tournament_size=3)
```

- `population` — list of rocket dicts
- `scores` — parallel list of floats (same order as population)
- `n_survivors` — how many to return
- `tournament_size` — how many candidates compete per tournament (default 3)

logic: repeat `n_survivors` times — pick `tournament_size` random indices, return the rocket with the highest score among them. return the full list of survivors.
